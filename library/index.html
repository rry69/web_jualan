Perpustakaan Simple

Sistem manajemen perpustakaan berbasis PHP Native yang memungkinkan pengelolaan buku, anggota, transaksi peminjaman, serta laporan.

📦 Fitur

  • 🔐 Autentikasi: Login, Register, Logout
  • 📚 Manajemen Buku: Tambah, Edit, Hapus, Upload gambar
  • 👤 Manajemen Anggota Perpustakaan
  • 🔄 Transaksi: Peminjaman dan Pengembalian Buku
  • 📊 Dashboard Ringkasan Aktivitas
  • 📁 Laporan: Ekspor laporan transaksi ke Excel
  • ⚙️ Manajemen Profil Admin

🛠️ Teknologi

  • PHP & MySQL
  • HTML, CSS, JavaScript
  • Bootstrap 5
  • jQuery & Chart.js
