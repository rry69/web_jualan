Sistem Inventaris Barang (PHP Native)

Aplikasi web berbasis PHP Native yang dirancang untuk mempermudah pelacakan dan pengelolaan barang dalam gudang atau organisasi. Sistem ini mendukung manajemen data barang, transaksi masuk dan keluar, laporan stok, serta pengelolaan pengguna dengan akses berbasis peran.

📋 Fitur

  • 📊 Dashboard: Ringkasan metrik jumlah barang, pergerakan harian, dan stok terkini.
  • 📦 Manajemen Barang: Tambah, edit, hapus, dan detail barang lengkap.
  • ➕ Barang Masuk: Pencatatan barang diterima dan pembaruan stok otomatis.
  • ➖ Barang Keluar: Validasi stok dan riwayat transaksi otomatis.
  • 📁 Laporan: Ekspor data transaksi dan stok untuk administrasi.
  • 👥 Manajemen Pengguna: Buat/edit/hapus pengguna dengan peran tertentu.
  • 🛡️ Akses Berbasis Peran:
    • Admin: Akses penuh termasuk manajemen pengguna
    • Petugas: Kelola barang dan transaksi
    • Viewer: Akses hanya-baca
  • 🖥️ Antarmuka: Minimalis, font Inter, sidebar modern dengan warna Midnight Blue, Sky Blue, dan Royal Blue.
  • 📱 Desain Responsif: Kompatibel untuk desktop dan mobile via AdminLTE.

🛠️ Teknologi

  • Backend: PHP 7.4+
  • Database: MySQL / MariaDB
  • Frontend: AdminLTE 3.2, Bootstrap 4.6, jQuery 3.6
  • Ikon: Font Awesome 5.15
  • Font: Inter (Google Fonts)
  • Lainnya: DataTables untuk tabel interaktif

💻 Persyaratan Sistem

  • Server: XAMPP (Apache & MySQL) atau sejenis
  • OS: Windows, Linux, atau macOS
  • PHP: Versi 7.4 atau lebih tinggi
  • MySQL: Versi 5.7+ atau MariaDB
  • Browser: Chrome, Firefox, Edge, atau Safari (terbaru)
  • Koneksi Internet: Untuk CDN selama pengembangan
